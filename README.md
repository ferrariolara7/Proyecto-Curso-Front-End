Proyecto Curso Front-End – Vigeo

Sitio web desarrollado como proyecto de curso Front-End, enfocado en una dietética y tienda de productos saludables llamada Vigeo.

📌 Descripción

El proyecto consiste en el desarrollo de una página web estilo ecommerce para una dietética, incluyendo distintas vistas y secciones informativas para mejorar la experiencia del usuario.

🚀 Tecnologías utilizadas
HTML
CSS
JavaScript
Git y GitHub

📂 Funcionalidades
Página de inicio
Vista “Conócenos”
Navegación mediante menú
Diseño responsive
Footer con información de contacto
Integración de íconos
Diseño moderno y limpio

🎯 Objetivo del proyecto
Aplicar conocimientos de desarrollo web frontend mediante la creación de un sitio funcional, visualmente atractivo y organizado.
